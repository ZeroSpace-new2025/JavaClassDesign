# AssemblyInfo.md - 程序集信息
## ProjectMain

---
### 1. 程序集概述
- **编写环境**: C# .NET 8 + Godot 4.x 2D 游戏引擎
- **层级定位**: 根节点（游戏入口），负责游戏生命周期管理与场景初始化
- **输入规范**: 接收 Godot 场景树事件、启动/保存/加载/退出指令
- **输出规范**: 提供游戏实例单例、场景树管理、游戏流程控制
- **依赖项**: Godot Engine API、Player（玩家模块）

### 2. 程序集封装功能
- **ProjectMain**
  - 游戏主入口类，继承 Godot Node2D，实现单例模式，负责游戏初始化、场景启动、存档管理与退出控制
